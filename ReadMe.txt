Chess.jar can be ran right off the desktop providing you have java .

When game begins Human player is to start by entering row and column and then move. The AI is fast which allows the Human player to enter row and columns as fast as they can.

Watch the top right hand corner for outputs such as computer is out of moves player wins. The areas where you enter row and columns will shorton too to suggest the game is over.

To exit the game, the player can hit the main panel red X and then run the game again from the chess.jar file if needed

if and when you Encounter a bug rerun the program which hopefully should not happen so offen.
